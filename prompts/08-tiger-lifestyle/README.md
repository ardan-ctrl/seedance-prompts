## Принципы продуктовой интеграции

**Из исследования (модель Nobody Sausage):**

1. Продукт — **последний штрих**, не решение проблемы. Тигр уже решил сам.
2. Тигр **никогда не говорит** о продукте. Только реакции.
3. Кот может взаимодействовать с продуктом — создаёт второй слой комедии.
4. Продукт входит в момент **уязвимости** (апатия, усталость, love-fail) — не в момент триумфа.
5. Максимум **2-3 секунды** с продуктом в кадре. Не больше.

## Позиции продукта в кадре

```
TOOL INTEGRATION    — продукт как инструмент работы (энергетик между клиентами)
PROP UPGRADE        — продукт заменяет generic предмет в обычной сцене
TWIST BEAT          — продукт появляется в финальной шутке как пуанта
VULNERABILITY MOMENT — продукт в момент love-fail / усталости / понедельника
```
