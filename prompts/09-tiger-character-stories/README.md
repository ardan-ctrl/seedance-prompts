## Зачем нужны эти ролики

Трансформационные видео строят охват. Но только если аудитория **влюблена в персонажа**.
Характерные сцены без "трансформации" строят эту любовь:

- **Тигр + Кот**: комедия отношений. Самая вирусная пара.
- **Тигр + Нора**: эмоциональный якорь. Аудитория видит уязвимость.
- **Love fail**: его единственная слабость. Делает идеального персонажа человечным.
- **Тигр один**: монологи, рефлексия, взгляд в камеру — прямое обращение к аудитории.

## Частота

```
Рекомендуемый ритм:
3-4 профессиональных трансформации → 1 характерная история
Или: каждая неделя — 2 трансформации + 1 характерная сцена
```
