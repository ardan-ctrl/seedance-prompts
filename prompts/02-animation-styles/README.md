## Как задавать стиль (первые слова промпта)

### Low-Poly 3D Chibi
```
[Style: low-poly 3D chibi animation, toy-like aesthetic, bright saturated colors,
smooth polygon surfaces, oversized head proportions, stiff blocky movement increments]
```

### Манга 1980-х
```
1980s–90s seinen manga ink — bold black lines, cross-hatching, screentone dots,
no color, no grey. Zero 3D, zero CGI. References: Akira/Ghost in the Shell.
```

### Клеймация
```
[Style: 3D claymation cinematic — soft clay texture on all surfaces,
visible fingerprint and tool marks on clay forms, smooth moldable physics]
```

### Клеймация + Киберпанк
```
[Style: 3D claymation cinematic — soft clay texture on every surface,
cyberpunk night environment with neon magenta/cyan/purple lighting,
deep slow-motion physics throughout except teleportation detonation]
```

### Mixed Media (2D + Live Action)
```
[Style: live-action mixed media — photorealistic real-world environment with real people,
single 2D flat cartoon character composited in, maintaining 2D aesthetic in 3D world]
```

### Bold 2D Аниме
```
[Style: bold 2D anime illustration, cel-shaded flat coloring, thick confident outlines,
warm amber and muted earth tones with cool electric blue holographic accents]
```
