## Ключевые техники для экшн

### Снайдер-ритм
```
RAMPS TO SLOW MOTION as [ключевой момент]
SNAPS BACK [внезапный возврат к полной скорости]
```

### Дестабилизированная POV камера
```
hyper-chaotic handheld motion, completely unstabilized, violent raw human movement,
constant micro-jitters, aggressive head swings, abrupt jerks, wide-angle lens
(strong distortion), subtle chromatic aberration near frame edges
```

### Голливудский монтаж (мультишот)
```
Montage, multi-shot action Hollywood movie, don't use one camera angle or single cut,
cinematic lighting, photorealistic, 35mm film quality, ARRI ALEXA aesthetic
```

### Single Shot (без монтажа)
```
Single continuous shot, no cuts, camera moves through the entire scene in one take
```
