## Аниме-жаргон для промптов

```
cel-shaded              — плоская заливка с чёткими тенями
speed lines             — линии скорости
impact stars            — звёзды удара
sweat drop              — капля пота (комедийный элемент)
screentone shading      — паттерн точек для теней (манга)
oversized expressive eyes
dynamic camera rotation
cherry blossoms scattered by energy
dramatic backlighting
fluid character animation
```
