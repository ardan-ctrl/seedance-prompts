## Формула эмоциональной арки

```
НАЧАЛО — персонаж в уязвимом/обычном состоянии
ВЫЗОВ — что-то меняется/угрожает/требует выбора
УСИЛИЕ — персонаж действует, не всё получается
ПОВОРОТ — неожиданное решение или открытие
ФИНАЛ — трансформация или принятие
```
