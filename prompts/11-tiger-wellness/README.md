# Tiger — Wellness & Stress Content

**Ключевая деталь:** Тигр управляет стрессом не ради здоровья. Ради внешности.
Стресс вызывает старение. Старение портит лицо. Лицо — это всё.

Это делает его:
- Комедийным (мотивация неожиданная)
- Аутентичным (мотивация узнаваемая)
- Идеальным амбассадором стресс-приложения

---

## Тональность

Тигр относится к управлению стрессом как к профессиональной обязанности.
"Я не медитирую для покоя. Я медитирую для своего лица."
Он делает это серьёзно, ритуально, с полным достоинством.
Побочный эффект — реальное благополучие — он никогда не упоминает.

## Связь с приложением (будущий проект)

Тигр будет амбассадором стресс-приложения.
Его история входа в продукт: он искал "anti-aging skincare routine" →
нашёл медитацию → стал евангелистом медитации по неправильной причине.
Теперь рекомендует всем — но всегда формулирует через внешность.

## Формула для wellness-контента

```
[0–2s] ХУК: тигр замечает что-то плохое (морщину, усталость, тёмные круги)
[2–8s] МОТИВАЦИЯ: стресс = старение = неприемлемо (его логика)
[8–13s] РИТУАЛ: тигр справляется со стрессом максимально серьёзно
[13–15s] ФИНАЛ: либо результат + кот не впечатлён / либо клиффхэнгер
```
